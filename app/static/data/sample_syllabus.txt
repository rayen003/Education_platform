Course Title: Introduction to Computer Science

Course Description:
This introductory course covers the fundamental concepts of computer science and programming. Students will learn basic programming principles, algorithmic thinking, and problem-solving strategies through practical coding exercises and projects.

Instructor: Dr. Jane Smith
Email: jane.smith@university.edu
Office Hours: Monday and Wednesday, 2:00 PM - 4:00 PM

Modules:
1. Introduction to Computing
   - History of Computing
   - Computer Architecture
   - Binary Number System
   - Boolean Logic

2. Programming Fundamentals
   - Variables and Data Types
   - Control Structures
   - Functions and Procedures
   - I/O Operations

3. Data Structures
   - Arrays and Lists
   - Stacks and Queues
   - Trees and Graphs
   - Hash Tables

4. Algorithms
   - Complexity Analysis
   - Sorting Algorithms
   - Searching Algorithms
   - Recursion

5. Object-Oriented Programming
   - Classes and Objects
   - Inheritance
   - Polymorphism
   - Encapsulation

6. Software Development
   - Software Development Life Cycle
   - Agile Methodologies
   - Version Control
   - Testing and Debugging

Assessment:
- Programming Assignments (40%)
- Midterm Exam (25%)
- Final Project (25%)
- Participation (10%)

Required Materials:
- "Python Programming: An Introduction to Computer Science" by John Zelle
- Access to a computer with Python 3.x installed

Learning Outcomes:
By the end of this course, students will be able to:
1. Understand fundamental programming concepts
2. Write and debug programs in Python
3. Apply algorithmic thinking to solve problems
4. Implement basic data structures
5. Understand object-oriented programming principles
